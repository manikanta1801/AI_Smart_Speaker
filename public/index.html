Getting Started

Setup Guide

From zero to DeepSeek-powered speaker in 6 steps.

1

Get Your Free DeepSeek API Key

Go to aistudio.google.com/app/apikey, sign in with Google, and click Create API Key. No credit card needed. You get 1,500 free requests per day.

2

Push Code to GitHub

Create a new repo at github.com/new called mi-deepseek-speaker. Push the project folder to it using git init, git add ., git commit -m "init", git push.

3

Deploy on Netlify

Go to netlify.com, click Add new site → Import from Git, select your repo. In Site configuration → Environment variables, add deepseek_API_KEY with your key. Click Deploy.

4

Create a Google Action

Go to console.actions.google.com. Create a new project → Custom → Blank. Set an invocation name like "my AI assistant". Under Fulfillment, set your webhook to: https://your-site.netlify.app/.netlify/functions/ai-webhook

5

Test in Actions Simulator

In the Actions Console, click Test. Use the simulator to say "Talk to my AI assistant". You should hear DeepSeek respond. Fix any issues before going live.

6

Use on Your Mi Speaker 🎉

Say "Hey Google, talk to my AI assistant" on your Mi Speaker. Ask any question, then say "goodbye" or "stop" when done. That's it — enjoy your DeepSeek-powered speaker!
